⚡ Power Automate 설정
📋 Power Automate 설정 방법 보기
1. make.powerautomate.com (Microsoft 365 로그인)
2. 새 흐름 → 즉시 클라우드 흐름 → 트리거: HTTP 요청을 받은 경우
3. 요청 본문 JSON 스키마:
{"type":"object","properties":{"fileName":{"type":"string"},"folderPath":{"type":"string"},"fileContent":{"type":"string"}},"required":["fileName","folderPath","fileContent"]} 4. 새 단계 → SharePoint 파일 만들기
  파일 콘텐츠(식): base64ToBinary(triggerBody()?['fileContent'])
5. 저장 → HTTP POST URL 복사 → 위에 붙여넣기
🔑 접속 코드 설정
📞 학교 전화번호 자동조회 (NEIS)
NEIS 교육정보 개방포털 API 키를 입력하고 조회하면 전체 학교 대표번호를 자동으로 가져옵니다.
조회 결과는 이 브라우저에 저장되어 학교 현황 팝업에 표시됩니다.
